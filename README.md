# Calculadora de Costos de Ingredientes
## Objetivo
Afianzar conocimientos y bases con reflex
## Funcionalidades objetivo
- Formulario para añadir ingrediente (nombre y costo)
- Una lista que muestra los ingredientes añadidos
- Un calculo en vivo que muestra el costo total
## Metas
Usar los 5 pilares de Reflex en un solo archivo
1. Componentes de UI (bloques de construccion de la web)
2. Estado (Datos)
3. Manejo de Eventos (funciones que realizan una accion luego de clickear algo)
4. Reactividad (actualizacion de pantalla al actualizar la lista de ingredientes)
5. Variables Computadas (calculo del costo total)
